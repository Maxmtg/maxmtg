### Incidencia Fácil

Plataforma SaaS para reportar, gestionar y realizar seguimiento de incidencias mediante códigos QR.

He diseñado y desarrollado el producto de extremo a extremo: arquitectura, interfaz, base de datos, seguridad, automatizaciones, pagos, despliegue y evolución funcional.

**Funcionalidades principales**

* Formularios públicos mediante QR sin necesidad de registro.
* Panel privado con vistas de lista y Kanban.
* Gestión multi-organización.
* Usuarios, roles y permisos.
* Seguridad mediante Row Level Security.
* Subida de fotografías.
* Seguimiento público seguro.
* Notificaciones por correo electrónico.
* Informes PDF y exportaciones.
* Integración de pagos y suscripciones con Stripe.
* Despliegue en Cloudflare Pages.

**Tecnologías**

`React` `JavaScript` `Vite` `Supabase` `PostgreSQL` `SQL` `Edge Functions` `Stripe` `Cloudflare` `Git`

### Crónicas del Cosmos

Plataforma automatizada de publicación y distribución de contenido digital.

El sistema recopila fuentes, selecciona candidatos editoriales, procesa y valida contenidos, publica en WordPress y distribuye automáticamente cada artículo en diferentes canales.

**Funcionalidades principales**

* Ingesta y normalización de fuentes RSS.
* Evaluación automática de candidatos editoriales.
* Integración de modelos de inteligencia artificial.
* Validación de contenido, estructura y SEO.
* Generación y preparación de imágenes.
* Publicación automática en WordPress.
* Distribución en Telegram, Facebook, Instagram y Threads.
* Aislamiento de errores por plataforma.
* Integración con Google Analytics y Search Console.
* Control de duplicados y frecuencia de publicación.

**Tecnologías**

`n8n` `WordPress` `JavaScript` `APIs REST` `JSON` `RSS` `Servicios de IA` `Cloudflare`

🌐 [cronicasdelcosmos.com](https://cronicasdelcosmos.com)

### OpsFlow Dashboard

Aplicación pública de portfolio para gestionar solicitudes operativas ficticias mediante un dashboard desarrollado con React y TypeScript.

El proyecto contiene código completamente público, pruebas automatizadas y despliegue continuo mediante GitHub Actions.

**Funcionalidades principales**

* Dashboard con métricas y actividad reciente.
* Búsqueda y filtros combinados.
* Vista de lista ordenable.
* Vista Kanban.
* Creación y edición de solicitudes.
* Cambio de estado, prioridad y responsable.
* Persistencia versionada mediante `localStorage`.
* Recuperación segura ante datos corruptos.
* Diseño responsive y accesible.
* Pruebas de dominio e interacción.
* Despliegue automático en GitHub Pages.

**Tecnologías**

`React` `TypeScript` `Vite` `Vitest` `React Testing Library` `CSS` `localStorage` `GitHub Actions`
